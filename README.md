# Kanban
Portfolio Kanban

before run: edit file ./src/config/firebase.js
```js
const config = {
  databaseURL: 'https://<projectId>.firebaseio.com', // edit with ur db url project
  projectId: 'projectId' // edit with ur project id
}
```

How to run:
```command
$ yarn
$ yarn serve
```

build:
```
$ yarn build
```
